# Flappy Bird

A fun, responsive browser-based Flappy Bird game built with HTML, CSS, and JavaScript.

## Play

Open `index.html` in any modern web browser.

## Controls

- Click or tap to make the bird fly
- Press `Space` or `Arrow Up` to jump
- Avoid the pipes and score as many points as possible

## Features

- Smooth canvas-based gameplay
- Responsive controls for desktop and mobile
- Score and high-score tracking
- High score saved with browser local storage
- Start, game-over, and restart screens

## Technologies

- HTML5 Canvas
- CSS3
- Vanilla JavaScript
